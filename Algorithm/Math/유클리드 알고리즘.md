# 유클리드 알고리즘

두 양의 정수, 혹은 두 다항식의 최대공약수를 구하는 방법이다. 

두 양의 정수 $a,\,b\;(a > b)$에 대하여 $a = bq + r\;(0 \le r < b)$ 이라 하면, $a,\,b$의 최대공약수는 $b,r$의 최대공약수와 같다. 

$$
    \gcd(a,\,b) = \gcd(b,\,r)
$$

즉, $r = 0$ 이라면, $a,\,b$의 최대공약수는 b가 된다.

## C/C++ 코드 
```C++
int gcd(int x, int y) { 
    if ( y == 0 ) return x ; 
    return gcd(y, x % y) ; 
}

// 숏코딩
int gcd(int x, int y) { return y ? gcd(y, x % y) : x ; }
```
## 예시 

12345와 1234 최대공약수 구하는 문제

$$
    12345 = 1234 ⋅ 10\,+\,5\\
    1234 = 5 ⋅ 246\,+\,4\\
    5 = 4 ⋅ 1\,+\,1\\
    4 = 1 ⋅ 4\\
    ∴ \gcd(12345, 1234) = 1
$$
